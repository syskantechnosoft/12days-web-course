Welcome to Angular Hands-on App!

This application demonstrates core Angular features:

  • Template-Driven Forms: Simple forms with `[(ngModel)]` and validation.
  • Reactive Forms: Complex forms with `FormBuilder` and custom validators.
  • Route Guards: Protected routes using functional `CanActivateFn`.
  • Public API Integration: Fetching data with `HttpClient` and `Signals`.

Use the navigation menu to explore!
